# 🍽️ Sajik Online Restaurant App

Sajik Online Restaurant App is a role-based restaurant management web application developed using **ASP.NET Core MVC**, **Entity Framework Core**, and **SQL Server**. The application provides an online platform where customers can browse food items, register/login, manage carts, and place orders, while administrators can manage restaurant operations through a secure admin dashboard.

The project follows the **MVC (Model-View-Controller)** architecture and implements **session-based authentication** with role-based authorization for Admin and Customer users.

---

## 🚀 Features

### 👤 Customer Module
- User Registration & Login
- Session-Based Authentication
- Browse Food Items by Category
- Search Functionality
- Cart Management
- Checkout Support
- Profile Management
- Order & Payment Tracking

### 🛠️ Admin Module
- Admin Dashboard
- Manage Categories
- Manage Food Items
- Manage Sliders/Banners
- View Customers
- Enable/Disable Customer Accounts

## 🔐 Authentication & Authorization

The application uses **session-based login management**.  
Two user roles are implemented:

- **Admin**
- **Customer**

Based on login role:
- Admin users are redirected to the Admin Dashboard
- Customer users are redirected to the User Home Page

---

## 📂 Project Architecture

The project is structured using:
- **Models** → Database entities & validations
- **Views** → UI pages using Razor
- **Controllers** → Business logic & request handling

Entity Framework Core Code-First approach is used for database operations and relationships.
